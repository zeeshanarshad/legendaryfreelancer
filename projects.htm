YouTube Captions Management System (2016 - 2020)

A custom web-based system was created for managing YouTube videos, captions, and auto-play with a timer. The project began from scratch with a single HTML text file containing a basic timer and was expanded over time through 100+ development phases into a complete, scalable platform.

Built with PHP/MySQL and AJAX/jQuery, the system supports multiple user logins for managing operations seamlessly. Additional modules included a custom quiz system, live timer, and advanced features, all structured with clear, well-commented code. This made it simple for the client to maintain and update with only basic knowledge of HTML, CSS, and JavaScript.

The result was a robust, user-friendly platform that grew organically from a simple idea into a full-scale management system, successfully running for a client in the USA.
